# Шрифти (self-hosted)

Покладіть сюди файли `.woff2` (сайт працює і без них — тоді використовується системний шрифт):

- `onest-600.woff2`, `onest-700.woff2`, `onest-800.woff2` — https://github.com/vjacheslav-bogdanov/onest (OFL)
- `inter-400.woff2`, `inter-500.woff2` — https://rsms.me/inter/ (OFL)

Субсет: latin + cyrillic (обовʼязково кирилиця!). Зручно згенерувати через https://gwfh.mranftl.com
(оберіть charset "cyrillic", формат woff2) або `pyftsubset`.
